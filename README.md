## ğŸš¦ Getting Started

### Prerequisites
- Node.js 18+
- PHP 8.2+
- DDEV
- Composer

### Development Setup

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd bizly
   ```

2. **Start the backend**
   ```bash
   cd backend
   ddev start
   ddev artisan migrate
   ```

3. **Start the frontend**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

4. **Access the application**
   - Frontend: http://localhost:3000
   - Backend API: https://bizly-backend.ddev.site
